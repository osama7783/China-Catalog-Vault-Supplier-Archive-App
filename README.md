# China Catalog Vault

China Catalog Vault is a web application designed to help organize, archive, and manage Chinese supplier information, product catalogs, business cards, and company records in one structured platform.

The project was built using modern web technologies, including React, TypeScript, Tailwind CSS, and Supabase as the backend database and authentication layer. It also includes an AI/OCR-based Smart Capture feature that can extract useful information from business cards or supplier documents, such as company names, phone numbers, emails, websites, WeChat accounts, and product-related details.

The main goal of this project is to solve a real business problem: managing supplier data collected from exhibitions, factories, online sourcing, and business communication in China. Instead of keeping scattered information in photos, chats, Excel files, or notebooks, the system provides a clean and searchable digital archive.

## Key Features

- Supplier and company database management
- Product catalog and category organization
- OCR/AI Smart Capture for business cards and documents
- Supabase database, authentication, and storage integration
- Responsive React-based web application
- Arabic-friendly interface for business users
- Searchable records for suppliers, contacts, catalogs, and notes
- Practical workflow for import/export and sourcing businesses

## Technologies Used
